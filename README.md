# git-starred-backend

Backend for the project. 
Made with Express.js

Handles authentication with GitHub, API requests and database stuff 

## To run
-
### install packages

yarn install 
or
npm install 

// run the server
yarn dev

dependencies:
express (Server)
prisma (ORM)
nodemon (run continously)
dotenv (read .env file)

used postgres database (Heroku)
